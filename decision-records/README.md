# Decision Records

## Objetivo
Registrar decisoes duraveis do AI Office para evitar rediscutir o mesmo criterio.

## Quando criar um registro
- Mudanca de fluxo oficial.
- Novo agente permanente.
- Nova biblioteca permanente.
- Quality gate que bloqueia entregas.
- Decisao tecnica que afeta varios projetos.

## Formato recomendado
- Titulo.
- Data.
- Contexto.
- Decisao.
- Consequencias.
- Arquivos afetados.

## Regras
- Registrar decisao, nao debate completo.
- Manter curto.
- Usar linguagem operacional.
- Atualizar se a decisao for substituida.
