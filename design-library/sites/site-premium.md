# Padrão de Sites Premium - Code a Code

Todo site institucional ou landing page deve:

- Ter hero section forte
- Ter imagem, vídeo ou ilustração de impacto
- Ter CTA claro acima da dobra
- Ter seções com storytelling
- Ter prova social
- Ter diferenciais visuais
- Ter animações suaves
- Ter hover states
- Ter scroll reveal
- Ter cards modernos
- Ter FAQ bem trabalhado
- Ter formulário ou botão WhatsApp
- Ter mobile excelente

## Estruturas possíveis

Nunca usar sempre a mesma ordem.

Variações permitidas:
- Hero + problema + solução + prova social + planos + FAQ
- Hero visual + serviços + processo + antes/depois + CTA
- Hero split + números + benefícios + galeria + depoimentos
- Hero full-screen + storytelling + cards animados + CTA fixo
- Hero com mockup + funcionalidades + comparação + CTA

## Banco de imagens

Usar imagens de:
- Unsplash
- Pexels
- Pixabay

Unsplash, Pexels e Pixabay oferecem APIs oficiais para buscar imagens/fotos; Unsplash tem API oficial de fotos/ilustrações, Pexels informa API gratuita de fotos/vídeos e Pixabay oferece API REST para imagens/vídeos royalty-free. :contentReference[oaicite:0]{index=0}
