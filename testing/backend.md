# Backend Testing

## Objetivo
Validar backend antes de concluir tarefa tecnica.

## Validar
- typecheck
- build
- variaveis de ambiente
- conexao real quando houver banco
- migrations e seed
- CRUD principal
- tratamento de erro

## Evidencias
- comandos executados
- logs principais
- banco usado
- limitacoes registradas

## Reprovacao
Falha de build, schema invalido, endpoint principal quebrado ou banco nao validado quando disponivel.
