# Performance Testing

## Objetivo
Validar desempenho minimo percebido e tecnico.

## Validar
- build de producao
- tamanho de bundle quando aplicavel
- imagens otimizadas
- consultas caras
- carregamento inicial
- resposta de endpoints criticos

## Evidencias
- comando executado
- metrica principal
- gargalo encontrado

## Reprovacao
Pagina ou endpoint central lento sem justificativa ou asset pesado sem otimizacao.
