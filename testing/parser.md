# Parser Testing

## Objetivo
Validar parser de PDF, CSV, XML ou arquivo externo.

## Validar
- pasta `samples/` com exemplos reais permitidos
- casos positivos
- casos com erro
- campos obrigatorios
- normalizacao de valores
- teste automatizado

## Evidencias
- arquivos sample usados
- comando de teste
- resultado esperado vs obtido

## Reprovacao
Parser sem sample real, sem teste automatizado ou aprovado apenas por inspecao manual.
