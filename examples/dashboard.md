# Example - Dashboard

## Objetivo
Exemplo curto para dashboards administrativos, financeiros ou operacionais.

## Fontes sugeridas
- `component-library/dashboard.md`
- `knowledge/frontend.md`
- `knowledge/backend.md` quando houver API
- `quality-gates/backend.md` quando houver banco
- `testing/api.md`

## Saidas esperadas
- indicadores definidos pelo Analista
- hierarquia visual definida por UX/UI
- fontes de dados mapeadas
- estados de loading, vazio e erro

## Validacoes
- totalizadores coerentes
- filtros por periodo quando aplicavel
- dados reais quando banco/API estiver disponivel
- build e typecheck
