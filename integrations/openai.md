# OpenAI

## Objetivo
Notas para projetos que usam APIs da OpenAI.

## Validar
- modelo adequado ao caso
- chave em variavel de ambiente
- tratamento de erro e timeout
- logs sem dados sensiveis
- custo/limite considerados

## Evidencias
- endpoint testado
- fallback ou mensagem de erro
- configuracao documentada

## Evitar
- hardcode de chave
- enviar dados sensiveis sem necessidade
- depender de modelo obsoleto sem revisao
