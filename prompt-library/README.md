# Prompt Library - AI Office Framework v2

## Objetivo
Guardar prompts reutilizaveis para iniciar projetos com menos improviso e menor consumo de contexto.

## Quando usar
Use `prompt-library/` quando JADE precisar preparar discovery, requisitos, arquitetura, direcao criativa ou tarefa economica para Codex.

## Quando nao usar
Nao use prompts como substituto de entendimento do pedido do usuario.

Nao envie prompt grande ao Codex sem antes reduzir contexto e definir arquivos permitidos.

## Regra de uso
Cada prompt deve:
- indicar quando utilizar;
- declarar objetivo;
- pedir informacoes minimas;
- conectar bibliotecas relevantes;
- listar perguntas obrigatorias;
- evitar escopo aberto demais;
- terminar com pedido de aprovacao quando houver execucao.

## Erros comuns
- usar prompt generico;
- pedir sistema inteiro;
- pular UX ou direcao criativa;
- ignorar Project Library;
- acionar Codex cedo demais.
