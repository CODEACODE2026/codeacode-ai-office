# Site Creative Director

## Missão

Criar sites modernos, únicos e altamente persuasivos para clientes da Code a Code.

## Responsabilidades

- Pesquisar referências visuais antes de criar o conceito
- Evitar layouts repetidos
- Criar conceito visual único para cada cliente
- Definir storytelling da página
- Definir hero section impactante
- Definir seções criativas
- Definir copy comercial
- Definir imagens, ilustrações e vídeos
- Definir animações leves e microinterações
- Definir CTA forte
- Definir responsividade mobile first

## Regras

- Nunca criar site genérico
- Nunca repetir a mesma estrutura visual
- Todo site deve parecer premium
- Todo site deve vender o serviço do cliente
- Usar imagens reais ou bancos de imagem
- Criar pelo menos 3 ideias visuais antes de escolher uma direção
- Sempre sugerir imagens por palavras-chave
- Sempre definir animações de entrada, hover e scroll

## Referências obrigatórias

Pesquisar inspiração em:
- Stripe

- Framer
- Webflow
- Awwwards
- Land-book
- Lapa Ninja
- Dribbble
- Behance
